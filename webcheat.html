Web Cheat Sheet

Handy CSS Rules that I might commonly use

Rule Name Description

font-family

Font Family

Allows you to specify the font for an element. You can get additional fonts from Google Fonts that can be linked directly to HTML.

font-family: "Roboto", Arial, sans-serif;

font-weight

Font Weight

Allows you to specify the weight (boldness) of the font for an element. This can be a numeric value (100-900) or a keyword (normal, bold).

font-weight: bold;

font-size

Font Size

Allows you to specify the size of the font for an element. This can be a length (in pixels, ems, etc.) or a percentage.

font-size: 16px;

text-align

Text Align

Allows you to specify the horizontal alignment of text within an element. Parameters include left, right, center, and justify. Justify will stretch the text to fill the width of the container, creating a clean edge on both sides.

text-align: justify;

text-decoration

Text Decoration

Allows you to specify the decoration of text for an element. This can be used to add an underline, overline, or line-through to text. You can also specify the color and style of the decoration.

text-decoration: underline;

line-height

Line Height

Allows you to specify the height of a line of text. This can be a unitless value (which is a multiplier), a length (in pixels, ems, etc.), or a percentage.

line-height: 1.5;

letter-spacing

Letter Spacing

Allows you to specify the spacing between letters in a text. This can be a length (in pixels, ems, etc.) or a percentage.

letter-spacing: 0.1em;

cursor

Cursor

Allows you to specify the type of cursor that should be displayed when the mouse is over an element.

cursor: pointer;

background

Background

Allows you to specify the background color or image for an element. You can use a solid color, a gradient, or an image as the background. Gradients can be linear or radial and can be created using the linear-gradient or radial-gradient functions. You can also specify the direction of the gradient and the colors to use.

background: -webkit-linear-gradient(-45deg, #0054b4, #141E30);

background: url('image.jpg') no-repeat center center;

background: linear-gradient(-45deg, #0054b4, #141E30);

color

Color

Allows you to specify the color of the text for an element. This can be a named color, a hexadecimal value, or an RGB value.

color: blue;

background-color

Background Color

Allows you to specify the background color for an element. This can be a named color, a hexadecimal value, or an RGB value.

background-color: lightblue;

-webkit-box-shadow

Box Shadow

Allows you to specify a shadow effect for an element.

-webkit-box-shadow: 0px 4px 10px 0pxrgba(0, 0, 0, 0.5);

-moz-box-shadow

Box Shadow

Allows you to specify a shadow effect for an element. (For Mozilla Firefox)

-moz-box-shadow: 0px 4px 10px 0px rgba(0, 0, 0, 0.5);

box-shadow

Box Shadow

Allows you to specify a shadow effect for an element. (Standard)

box-shadow: 0px 4px 10px 0px rgba(0, 0, 0, 0.5);

max-width

Max Width

Allows you to specify the maximum width of an element.

max-width: 500px;

margin

Margin

Allows you to specify the space around an element.

margin: 10px;

padding

Padding

Allows you to specify the space inside an element.

padding: 10px;

border-radius

Border Radius

Allows you to specify the radius of the corners of an element.

border-radius: 10px;

opacity

Opacity

Allows you to specify the transparency of an element. Values range from 0 (fully transparent) to 1 (fully opaque).

opacity: 0.5;

transition

Transition

Allows you to specify the duration and timing function of a transition effect.

transition: opacity 0.3s ease-in-out;

Flexbox

Property Name Description

flex-grow

Flex Grow

Allows you to specify how much an element should grow relative to its siblings when there is extra space available.

display: flex;
flex-grow: 1;

flex-shrink

Flex Shrink

Allows you to specify how much an element should shrink relative to its siblings when there is not enough space available.

display: flex;
flex-shrink: 1;

flex-basis

Flex Basis

Allows you to specify the initial size of an element before it starts to grow or shrink.

display: flex;
flex-basis: 200px;

justify-content

Justify Content

Specifies the alignment of flex items along the main axis. Justify content can have the values of:

  • flex-start: aligns items to the start of the container
  • flex-end: aligns items to the end of the container
  • center: centers items in the container
  • space-between: distributes items evenly with the first item at the start and the last item at the end of the container
  • space-around: distributes items evenly with equal space around them
  • space-evenly: distributes items evenly with equal space between them

display: flex;
justify-content: center;

Useful Semantic HTML5 Elements

Element Name Description

<header>

Header

Defines a header for a document or section. It typically contains introductory content or navigational links.

<main>

Main

Defines the main content of a document. It should contain content that is unique to the document and not repeated across other documents (like sidebars, navigation links, etc.).

<footer>

Footer

Defines a footer for a document or section. It typically contains information about the author, copyright information, links to related documents, etc.

<nav>

Nav

Defines a set of navigation links.
